# Candidate queries for retrieval fixtures

Atomic, single-intent queries harvested from production LangSmith traces
(project `isundaylee-istaroth-web`, via `scripts/langsmith_tools.py
list-queries`), captured 2026-06-21. These are candidates for new
`retrieval_fixtures/` cases; each still needs verbatim corpus ground truth
derived per `.agents/skills/retrieval-fixtures/SKILL.md` before it becomes a
fixture.

Frequency is intentionally NOT recorded — each distinct query is treated
equally. Excluded from the list: proper-noun popup queries (`“X”是什么？` /
`What is “X”?`, machine-generated by
`frontend/src/hooks/useProperNounSelection.tsx`) and the raw internal dialog-id
query `Cs_NK_AQ603610_BYS_CHS 说的是什么`. English queries are folded into their
CHS equivalents.
